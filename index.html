A list of accessibility features

  • Tab Control: Help users who rely on the keyboard to navigation, as well as screen reader users
  • Keyboard shortcuts for Dialer and Tabs: Improve user experience
  • Accessible Table: Help screen readers to navigate formatted tables
  • Help Tab: Provide user manual-like information
  • Captioned Images: Primarily useful for screen reader users
  • ARIA landmark roles: Help assistive tech know how to navigate through the app
  • ARIA live regions: Provide access to dynamic web applications
